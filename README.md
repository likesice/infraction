# INFRACTION

A simple web-application to split expenses with friends

## Purpose

While there are quite a few expense splitting apps out there, they charge
a subscription fee for features that are quite basic. 

Infraction aims to replicate the following features of these apps:

- User Authentication/Authorization
- Creation of Events/Groups ("Infractions") 
- Calculating owed amounts between members of an infraction

Infractions does not aim to implement any of the following:

- Handling Payments
- Anything else...
